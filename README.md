# 🏥 Hospital Management Platform

A production-grade Hospital Management Platform built to demonstrate modern DevOps practices, cloud infrastructure automation, and Kubernetes deployment on AWS.

## Project Goals

- Build and deploy a full-stack application
- Automate infrastructure with Terraform
- Containerize applications using Docker
- Orchestrate workloads with Kubernetes (Amazon EKS)
- Implement CI/CD using GitHub Actions
- Monitor applications using Prometheus and Grafana
- Centralize logs with CloudWatch
- Follow production-grade DevOps best practices
